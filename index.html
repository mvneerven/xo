ExoForm

Next Level Forms Generation

XO-JS is a compact, developer-friendly, pure JavaScript (ES6) Component Library that helps with Progressive Web App development and includes a Declarative Web Forms Engine

Foundations

The XO-JS components are the result of a lifetime experience in Web Development, a passion for clean, high performing JavaScript and a continuous quest for the perfect SPA/PWA.

Marc van Neerven has developed an award winning CMS (Smartsite), written a W3C XForms Compliant Forms Engine, and is the author of the 'Naked SPA' blog series on Medium.com.

Features

  • Declarative JSON syntax
  • Single & Multiple page forms
  • Wizards, Surveys, custom paging
  • Typed, nested POST data for easy REST/GraphQL calls
  • Semantic HTML, CSS3 Grid
  • HTML5 validations, plus any custom validation
  • Extremely Developer-Friendly (see for yourself)
  • All standard HTML5 controls, plus many more
  • CKEditor, ACE Editor, MonacoEditor included
  • Flexible and Extensible
  • Vanilla JavaScript (ES6)
  • No dependencies
  • Open Sourced on GitHub & NPM
  • Builder environment included
  • Easy integration in Vue, Angular, React, etc.
  • CodePen examples available
