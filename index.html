ExoForm

Next Level Forms Generation

ExoForm is an ES6 JavaScript Framework for complex HTML Forms.

Experience

Based on earlier work in Dynamic Form Generation
Such as a W3C Compliant XForms Engine

Features

  • Declarative JSON syntax
  • Single & Multiple page forms
  • Wizards, Surveys, custom paging
  • Typed, nested POST data for easy REST/GraphQL calls
  • Semantic HTML, CSS3 Grid
  • HTML5 validations, plus any custom validation
  • Extremely Developer-Friendly (see for yourself)
  • All standard HTML5 controls, plus many more
  • CKEditor, ACE Editor, MonacoEditor included
  • Flexible and Extensible
  • Vanilla JavaScript (ES6)
  • No dependencies
  • Open Sourced on GitHub & NPM
  • Builder environment included
  • Easy integration in Vue, Angular, React, etc.
  • CodePen examples available
